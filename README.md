# 롯데월드타워 스탑워치

롯데월드타워 야경을 테마로 한 웹 스탑워치입니다.

## 미리보기

![롯데월드타워 스탑워치](https://github.com/sitacule/test/raw/master/preview.png)

## 기능

- 시작 / 정지
- 랩 타임 기록 (최고 랩 초록색, 최저 랩 빨간색 표시)
- 초기화
- 10ms 단위 정밀 측정

## 디자인 특징

- 롯데월드타워 실루엣 (clip-path 기반)
- 창문 랜덤 점등 애니메이션
- 서울스카이 전망대 발광 링
- 별 반짝임 야경 배경
- 석촌호수 물 반사 효과

## 사용 방법

별도 설치 없이 `stopwatch.html` 파일을 브라우저에서 열면 바로 실행됩니다.
